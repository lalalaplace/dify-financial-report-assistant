# 系统架构

本项目由三层组成：数据抽取层、结构化存储层和智能问数层。

## 数据抽取层

Python 脚本负责扫描财报 PDF、识别公司和报告期、定位资产负债表、利润表和现金流量表，并将表格块转换为结构化 JSON。

## 结构化存储层

PostgreSQL 保存公司维度表、财报索引表、字段字典表、中间抽取结果表和最终标准财务表。最终问数主要查询 `balance_sheet`、`income_sheet`、`cash_flow_sheet`。

## 智能问数层

Dify 负责自然语言理解、SQL 生成、安全审查和结果解释。FastAPI 提供只读 SQL 查询接口，并通过表名白名单和关键字拦截降低误操作风险。
